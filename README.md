# FitsMe

**FitsMe** is an app that takes the ***hassle*** out of online clothes shopping. FitsMe partners with women's fashion retailers to recommend the perfect size every time and reduce returns *as well as* the negative environmental impact of package returns.

Our **SizeFinder** quiz is ***quick*** and **easy**.

**FitsMe** is the final project built by 4 students enrolled in the Full Stack Web Development bootcamp at Le Wagon Berlin.
This web application is built with Ruby on Rails as well as Ruby, HTML, CSS, & JavaScript.

## Install

### Clone the repository

```shell
git clone git@github.com:svenja-tech/Fitsme.git
cd project
```

### Check your Ruby version

```shell
ruby -v

### Install dependencies

Using [Bundler](https://github.com/bundler/bundler) and [Yarn](https://github.com/yarnpkg/yarn):

```shell
bundle && yarn
```

### Initialize the database

```shell
rails db:create db:migrate db:seed
```

### Add heroku remotes

Using [Heroku CLI](https://devcenter.heroku.com/articles/heroku-cli):

```shell
heroku git:remote -a project
heroku git:remote --remote heroku-staging -a project-staging
```

## Serve

```shell
rails s
```
